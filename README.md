### Отчет по второй практической работе

В рамках второй практической работы мне удалось запустить нейронную сеть с использованием библиотеки PyTorch, разобраться в ее структуре, а также собрать другую нейронную сеть с использованием этой же библиотеки.

Созданная нейросеть представляет собой простую полносвязную модель с одним скрытым слоем. Она обучается на случайных данных, используя стохастический градиентный спуск (SGD) и функцию потерь MSELoss. В ходе работы реализованы этапы прямого и обратного проходов, а также сохранение обученной модели.
